Welcome to the Brightness Test! (Daylight and 50% brightness)

In this test, several images will be presented to you. In each image, you see combination of three digits (for example 1 5 3), shown with diffrent brightness. Simply enter the three numbers that you have seen on the answer box below each audio file, and click on "Next". The test will stop automatically.

You should do this task while there is daylight and in room with good lighting condition and the screen brightness set to 50%.

Impotant:
1- Please use a computer to participate in the tasks
2- After finishing each task a file will be generated. Please save it on your computer
3- After finishing all tasks please send all files to this Email-address: a.beshtawi@tu-berlin.de
4- Follow the instructions of each task to help us getting the best result.
5- If you could not recognise the digits in an Image write any number in the text box and click Next.
